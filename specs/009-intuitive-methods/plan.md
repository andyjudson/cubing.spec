## Summary

Add a new static Intuitive Methods page to `cfop-app` with two pedagogical sections (Intuitive Cross and Intuitive F2L), reusing legacy-approved text and image assets while applying clarified rules: keep missing-image cards visible with placeholders, remove creator-specific references, display only complete move hints, and keep this feature non-interactive.

**Language/Version**: TypeScript (~5.9), React 19  
**Primary Dependencies**: React Router DOM, Bulma, React Icons (existing stack only)  
**Storage**: Static assets from `cfop-app/public/assets/cfop_bgr` (no DB)  
**Testing**: Manual responsive verification + `npm run build`  
**Target Platform**: Web (desktop + iOS/mobile Safari baseline)  
**Project Type**: Single frontend SPA (`cfop-app`)  
**Performance Goals**: Fast static render, no added runtime fetches for page content  
**Constraints**: Content-first planning; neutral wording; static-only scope; no new dependencies  
**Scale/Scope**: One new route/page + nav entry + static instructional sections
